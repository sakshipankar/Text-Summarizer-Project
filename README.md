## How to run?

### STEPS:

### Clone the repository 
```bash 
https://github.com/sakshipankar/Text-Summarizer-Project
```
### Step 01 - Create  Virtual environment 
```bash 
python -m venv venv

.\venv\Scripts\activate 
```

### Step 02 - Install the requirements
```bash 
pip install -r requirements.txt
```

### run 
``` bash 
python app.py
```
